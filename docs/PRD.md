# 랜덤 산책 미션 생성기 — Product Requirements Document

**버전:** 1.0  
**작성일:** 2026-06-23  
**작성자:** 인터뷰 기반 정리  

---

## 1. 제품 개요

### 1.1 제품명
랜덤 산책 미션 생성기

### 1.2 문제 정의
운동하기 귀찮고 산책이 지루하다는 심리적 장벽이 존재함.

### 1.3 해결 아이디어
산책을 **게임으로 바꾼다**. 랜덤 미션을 부여해 일상 산책을 현실판 포켓몬GO처럼 탐험 경험으로 전환.

### 1.4 예시 미션
- 빨간 간판 5개 찾기
- 강아지 3마리 보기
- 편의점 2곳 찍기
- 나무 10그루 지나가기

## 3. 핵심 기능 요구사항

### 3.1 사용자 인증
- **방식:** 닉네임 입력만으로 시작 (회원가입 없음)
- **저장:** 브라우저 localStorage

### 3.2 플레이 방식

| 구분 | 혼자 걷기 (Solo) | 함께 걷기 (Group) |
|------|-----------------|-----------------|
| 미션 완료 | 자기 선언 | 사진 인증 (공식 기록) |
| 입장 방식 | 닉네임만 | 4자리 방 코드 |
| 모드 선택 | 없음 (일반 미션 풀 고정) | 없음 (경쟁 모드 고정) |
| 랭킹 | 없음 | 이동 거리 + 미션 완수 시간 기준 |
| 결과 표기 | 선언 뱃지 | 인증 뱃지 (구분 표기) |

> **결정 사항:** 혼자 걷기는 모드 선택 없이 바로 난이도 선택으로 진행. 함께 걷기는 방 코드 입장 후 경쟁 모드로 고정 시작.

### 3.4 난이도

| 난이도 | 미션 개수 | 설명 |
|--------|-----------|------|
| 쉬움 | 3개 | 가볍게 동네 한 바퀴 |
| 보통 | 5개 | 적당히 땀 흘리기 |
| 어려움 | 10개 | 진정한 탐험가의 도전 |

### 3.5 미션 생성 방식

#### A. 랜덤 뽑기
- 50개 사전 제작 미션 풀에서 난이도에 맞게 무작위 선택
- 모드별 미션 풀 구분 (일반/커플/친구/가족)

#### B. AI 미션 (ChatGPT API)
- 사용자가 현재 위치(동네명)를 입력
- ChatGPT API (gpt-4o-mini) 호출로 위치 맞춤 미션 즉석 생성
- API 키는 사용자 직접 입력 (sk-...)
- API 키 미입력 시 랜덤 뽑기 전용으로 사용 가능

### 3.6 GPS 거리 측정
- 기기 내장 Geolocation API 사용 (`watchPosition`)
- Haversine 공식으로 이동 거리 계산
- GPS 스파이크 필터: 0.1km 이상 순간 이동 무시
- 실시간 거리 + 경과 시간 화면 표시

### 3.7 사진 인증
- 미션 카드에서 📸 사진 인증 버튼 → 카메라 직접 호출
- File API + FileReader로 base64 변환 후 상태 저장
- 인증된 미션은 결과 화면에서 썸네일 표시

### 3.8 뱃지 & 칭호 시스템 (누적 거리 기반)

| 누적 거리 | 이모지 | 칭호 |
|-----------|--------|------|
| 0km~ | 🌱 | 새싹 탐험가 |
| 50km~ | 👟 | 동네 산책러 |
| 100km~ | 🗺️ | 도시 탐험가 |
| 200km~ | ⛰️ | 걷기 마스터 |
| 300km~ | 🌟 | 산책 레전드 |
| 400km~ | 👑 | 산책 왕 |
| 500km~ | 🏆 | 전설의 산책러 |

- 누적 거리는 localStorage에 영구 저장 (`walk_total_km`)
- 홈 화면에서 현재 등급 + 다음 등급까지 남은 거리 표시

### 3.9 결과 공유
- Web Share API 우선 사용 (모바일 네이티브 공유)
- 미지원 시 클립보드 복사 fallback
- 클립보드도 실패 시 prompt() 텍스트 선택 fallback
- 공유 내용: 닉네임, 뱃지, 이동거리, 미션 완수 수, 해시태그

### 3.10 산책 취소
- 산책 중 취소 버튼 제공
- 취소 시 기록 저장 없이 미션 선택 화면으로 복귀
- 확인 다이얼로그로 실수 방지

## 4. 화면 구성 (9개 스크린)

```
환영 화면
  ↓
혼자/함께 선택
  ├─ 혼자 걷기 → 난이도 선택 (모드 선택 없음)
  │               ↓
  │           미션 생성 방식
  │               ├─ 랜덤 뽑기 → 산책 진행
  │               └─ AI 미션 → 위치 입력 → AI 생성 → 산책 진행
  │
  └─ 함께 걷기 → 방 코드 입력/생성 → 방 대기실 (멤버 목록)
                                         ↓
                                     경쟁 모드 고정 → 난이도 선택
                                         ↓
                                     산책 진행 (동시 출발)
                                         ↓
                                     결과 화면 (랭킹 테이블)
```

## 5. 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 구조 | 단일 HTML 파일 | 배포 불필요, 파일 공유만으로 사용 |
| 스타일 | Vanilla CSS + CSS Variables | 외부 의존성 최소화 |
| 스크립트 | Vanilla JavaScript ES6+ | 프레임워크 없이 단순 유지 |
| 위치 | Geolocation API | 브라우저 내장, 별도 SDK 불필요 |
| 사진 | File API + FileReader | 브라우저 내장 |
| 공유 | Web Share API | 모바일 네이티브 공유 지원 |
| 저장 | localStorage | 서버 없이 영구 저장 |
| AI | OpenAI ChatGPT API (gpt-4o-mini) | 위치 맞춤 미션 생성 |
| 폰트 | Plus Jakarta Sans (Google Fonts) | 디즈니 감성 둥근 서체 |

**타이포그래피:**
- 폰트: Plus Jakarta Sans (Google Fonts CDN)
- 헤드라인: ExtraBold(800), letter-spacing: -0.03em
- 본문: Medium(500), line-height: 1.75
- 보조 레이블: Bold(700), uppercase, letter-spacing: 0.06em

**형태:**
- 기본 반경: 18px (카드, 컨테이너)
- 소형 반경: 12px (입력창, 작은 카드)
- 버튼: pill형 (999px)

**그림자 (warm-tinted):**
- 카드: `0 2px 16px rgba(120,80,60,0.10), 0 1px 4px rgba(120,80,60,0.06)`
- 소형: `0 1px 8px rgba(120,80,60,0.08)`
- 호버 리프트: `0 6px 24px rgba(201,104,69,0.16)`

**컴포넌트 규칙:**
- **선택 버튼**: 흰 카드(`#FFFDFB`) + 왼쪽 4px 컬러 보더 (풀컬러 배경 금지)
- **진행 바**: 단색 Primary (`#C96845`), 그라디언트 금지
- **미션 체크**: 완료 시 세이지 그린 채운 원 + 흰색 체크마크; 미완료 시 빈 회색 원
- **뒤로 버튼**: Surface Low 배경 + 1px 아웃라인 + Tabler `ti-arrow-left` 아이콘
- **버튼 색상 구분**:
  - 주요 액션 (사진 인증, 시작하기): Primary 테라코타
  - 산책 종료하기: Accent 앰버 골드
  - 보조 액션 (다시 산책하기, 입장): Secondary 세이지 그린
- **아이콘**: Tabler Icons Webfont (`@tabler/icons-webfont@3.31.0`), outline 스타일
- **화면 전환**: `translateY(12px) → 0` fade-in, `cubic-bezier(0.22, 1, 0.36, 1)`, 240ms

## 7. 미션 풀 구성 (50개 예시)

### 일반 모드 (37개)
찾기/관찰 미션, 이동/행동 미션, 촬영/재미 미션으로 구성.  
예) 빨간 간판 5개 찾기, 강아지 3마리 보기, 우체통 찾아서 사진 찍기, 공공 화장실 찾기, 하늘 사진 찍기 등

### 커플 모드 (15개)
함께하는 감성 중심.  
예) 손잡고 횡단보도 건너기, 같이 셀카 찍기, 벤치에서 5분 함께 쉬기 등

### 친구 모드 (15개)
경쟁/협동 게임 요소.  
예) 가위바위보 해서 진 사람이 사진 찍기, 가장 웃긴 간판 찾기 경쟁 등

### 가족 모드 (15개)
어린이 친화, 자연 탐색 중심.  
예) 개미 찾기, 구름 모양 동물 찾기, 나뭇잎 5개 모아서 사진 찍기 등

## 8. 함께 걷기 (Group) 상세

- **방 코드:** 4자리 숫자 코드로 방 생성/입장
- **저장 방식:** localStorage (`walk_room_XXXX`) — 같은 기기 데모용
- **모드:** 경쟁 모드 고정 (모드 선택 화면 없음)
- **난이도 선택:** 방장이 방 대기실에서 선택, 멤버 전원에게 동일 적용
- **대기실:** 멤버 목록 + 준비 상태 표시, 방장이 "지금 시작하기" 버튼으로 출발
- **같은 미션:** 방장이 설정한 미션을 멤버 전원이 동일하게 수행
- **미션 완료:** 사진 인증만 공식 기록으로 인정
- **랭킹 기준:** 이동 거리 → 미션 완수 시간 순
- **결과 화면:** 멤버별 거리/미션 수/시간 랭킹 테이블 표시
